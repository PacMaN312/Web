# VocabApp

Статичний Web-сайт для вивчення іноземних слів  
(режим навчання та контроль знань)

## Мета роботи

Ознайомитись із засобами HTML5 та CSS3 та навчитись застосовувати фреймворк Bootstrap для швидкої побудови Web-інтерфейсу користувача.

## Використані технології

- HTML5
- CSS3
- Bootstrap 5

## Структура проєкту

vocab-app/
│
├── index.html          (Про додаток)
├── register.html       (Реєстрація користувача)
├── login.html          (Вхід до сайту)
├── profile.html        (Профіль користувача)
├── study.html          (Робоча сторінка)
└── css/
    └── style.css       (Додаткові стилі)

## Опис сторінок

### 1. Про додаток (index.html)
Містить короткий опис системи, логотип та опис режимів роботи.

### 2. Реєстрація (register.html)
Форма з полями:
- Ім’я
- Email
- Стать
- Дата народження

### 3. Вхід (login.html)
Форма авторизації:
- Email
- Пароль

### 4. Профіль користувача (profile.html)
Відображення даних користувача у табличному вигляді.

### 5. Робоча сторінка (study.html)
Містить:
- Режим вивчення слів
- Блок контролю знань (тест)

## Адаптивність

Сайт адаптивний та коректно відображається на:
- Desktop
- Tablet
- Mobile

Використано Bootstrap Grid System та принцип Mobile First.

## Зауваження

Сайт є статичним.
Функціональність (обробка форм, збереження даних) не реалізована відповідно до вимог завдання.
